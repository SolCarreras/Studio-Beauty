# 💇 Studio Beauty - Web App

Aplicación web para una peluquería moderna desarrollada con:

- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQL Server

## ✨ Funcionalidades

- Landing page moderna
- Sección de servicios
- Formulario de reservas
- Backend con Express
- Integración futura con base de datos SQL Server
- Reservas online

## 📁 Estructura del proyecto

```bash
peluqueria/
│
├── backend/
├── frontend/
├── static/
├── package.json
└── README.md
```

## 🚀 Tecnologías utilizadas

- Node.js
- Express
- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo del proyecto

Este proyecto fue desarrollado como práctica de desarrollo fullstack,
simulando una aplicación real para gestión de reservas en una peluquería.
